CertWare Exports

Identifier:
net.certware.export.net.certware.export

Since:
1.1.0

Description:
Provides export capabilities to the run-time platform. Installs the exports for selection through standard workbench operations.

Configuration Markup:

<!ELEMENT extension (styleIdMapping | styleResource | styleContribution | exportOperation)+>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED

>


<!ELEMENT styleResource EMPTY>

<!ATTLIST styleResource

styleTheme CDATA #IMPLIED

styleFile  CDATA #REQUIRED

>

An XML resource file associated with the example to be copied into the workspace.


  • styleTheme - Description of the nature style file.
  • styleFile - An XML style file to be loaded prior to export operations.

<!ELEMENT styleIdMapping EMPTY>

<!ATTLIST styleIdMapping

modelElementID   CDATA #REQUIRED

styleString      CDATA #REQUIRED

isParagraphStyle (true | false)

>

Maps one of the built-in style IDs to a model element or attribute. Use the model's package identifier for the identifier value. Use the style IDs from the built-in document styles (see docx4j styles.xml), e.g. "Heading1" for a paragraph style or "Heading1Char" for a run style.


  • modelElementID - Model element number from the model package file.
  • styleString - Paragraph or run style ID from built-in or supplemental styles.
  • isParagraphStyle - True if the style is for paragraphs, false if style is for runs.

<!ELEMENT styleContribution EMPTY>

<!ATTLIST styleContribution

styleString      CDATA #REQUIRED

isParagraphStyle (true | false)

>

Provides an XML-syntax string to define a new style.


  • styleString - XML string describing the style including the namespace. The style name should be included in the string as it will be retrieved for application during the export process. The usual namespace for word processing documents in this framework is xmlns:w="http://schemas.openxmlformats.org/wordprocessingml/2006/main".
  • isParagraphStyle - True if the style is for paragraphs, false otherwise (for runs).

<!ELEMENT exportOperation EMPTY>

<!ATTLIST exportOperation

fileExtension CDATA #REQUIRED

class         CDATA #REQUIRED

>

Class for exporting the argument as a formatted document from the export wizard.


  • fileExtension - The model file extension to match when running the export wizard.
  • class - The export operation class will be called from the workbench export wizard when a file of this type is selected for export.
