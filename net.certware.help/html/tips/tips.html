CertWare Tips

  1. Use the "new" wizards to create a new ARM, AML, GSN, CAE, or EUROCONTROL argument model.
  2. There are two types of models available for some argument styles: a diagram model type and a structured tree model type.
  3. Use the examples wizard to populate a project with predefined safety case artifacts.
  4. Save diagram image fragments to the workspace as resources, then link to them in wikitext documents.
  5. Manage the safety case artifacts with the EGit team provider.
